A terminal-styled ASCII art generator with a retro hacker vibe. Type some text, pick a font, get ASCII art.

**Live:** https://ascii.lazylab.page/

## What it does

- Convert text to ASCII art
- 10 different fonts to choose from
- Copy to clipboard
- Looks like a terminal from the 90s

## Tech

Just one HTML file. Uses [Figlet.js](https://github.com/patorjk/figlet.js) for the ASCII generation.

## How to use

Open `index.html` in a browser. That's it.
